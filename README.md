# weather_web_app
 This is weather flask app.
 
 
 You can clone this code and use it anywhere. 
 
 created using flask in python and used jinja templates.
 
 Used API of weather details from openweathermap website. ( https://openweathermap.org/ )
 
 in order to use this code. 
 
 just download the code or clone it and use it using your own API key.
 
 <<<<  replace the API key in app.py with your API key.  >>> 
 make sure index.html file is in templates folder and style.css in static/css/style.css
 
 
 live demo - ( http://2thless.pythonanywhere.com/ )
